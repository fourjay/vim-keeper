vim-keeper
==========

Simple integrated documentation viewer inspired by investigate
It is intended to work with a browser that outputs text, but it
will call an external graphical utility.

## Features

* Useful out of the box
* Context sensitive
* Help is in buffer. Cut and paste

## Usage

Two mappings are provided

`KK` does a context sensitive search for documentation
`:WebMan` does a context sensitive lookup on an arbitrary keyword
